### TF-IDF inverted index

## Dependencies

Install Dependencies via ```pip3 install -r requirements.txt```  

- pandas==0.22.0
- numpy==1.14.0
- pymystem3==0.1.9
- scikit_learn==0.19.1

## Usage

```
usage: index.py [-h] [-l] [-g] [-o OUTPUT] input query

positional arguments:
  input                 JSON with Rambler news
  query                 Query to find similar news

optional arguments:
  -h, --help            show this help message and exit
  -l, --lemmatize       Use lemmatization
  -g, --ngrams          Use ngrams
  -o OUTPUT, --output OUTPUT
                        Numbers of docs to show
```
